**AI Powered Cricket Coaching System**             
An AI-powered cricket analysis platform that uses Video-based RAG and pose detection to evaluate player performance and generate personalized coaching feedback by comparing player movements with expert techniques.       

**Features**             
Cricket Practice Video Analysis            
AI-based Personalized Coaching Feedback        
Pose Detection and Movement Tracking            
Comparison with Expert Technique Benchmarks                
Video-based Retrieval-Augmented Generation (RAG)                
Performance Evaluation and Insights                 
Real-time Motion Analysis
